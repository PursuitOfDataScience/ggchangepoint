# WBS2 wrapper — Wild Binary Segmentation 2

Wraps the `breakfast` package. Requires the `breakfast` package.

## Usage

``` r
wbs2_wrapper(x, ...)
```

## Arguments

- x:

  A numeric vector.

- ...:

  Additional arguments passed to
  [`breakfast::breakfast()`](https://rdrr.io/pkg/breakfast/man/breakfast.html).

## Value

A `ggcpt` object.
