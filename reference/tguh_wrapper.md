# TGUH wrapper

Wraps the `breakfast` package for Tail-Greedy Unbalanced-Haar detection.

## Usage

``` r
tguh_wrapper(x, ...)
```

## Arguments

- x:

  A numeric vector.

- ...:

  Additional arguments passed to
  [`breakfast::breakfast()`](https://rdrr.io/pkg/breakfast/man/breakfast.html).

## Value

A `ggcpt` object.
