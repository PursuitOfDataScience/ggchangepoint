# FMS (Four-Metric-Segments) test signal

A piecewise-constant test signal from the WBS/NOT literature.

## Usage

``` r
signal_fms(n = 2000, seed = NULL)
```

## Arguments

- n:

  Length of the signal. Defaults to 2000.

- seed:

  Optional seed.

## Value

A tibble with columns `index` and `value`.
